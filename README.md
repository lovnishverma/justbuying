# 🌟 Overview

JustBuying is a complete WooCommerce-based e-commerce platform developed using WordPress with a fully customized storefront.

The project was built to provide a modern online shopping experience with optimized performance, secure payment workflows, mobile responsiveness, and scalable architecture suitable for production deployments.

This repository is maintained as a professional portfolio demonstrating full-stack WordPress and WooCommerce development.

# 🔒 Security

Sensitive production assets are intentionally protected.

Excluded information includes:

* Production credentials
* Database passwords
* API Keys
* SMTP configuration
* Environment variables
* Payment gateway secrets
* Customer information

The backup archives contained in this repository are password protected.

Passwords are **not** distributed through GitHub.

---

# 📦 Backup

The project backup is split into multiple password-protected archives.

```
part1.zip
part2.zip
part3.zip
```

These archives contain the complete production backup generated using **UpdraftPlus Backup/Restore**.

Refer to **installation.md** for restoration instructions.
